Agent Analytics 是一套给代理使用的 Web Analytics，同一套分析界面通过 plugin、skill、MCP、CLI 和 HTTP API 暴露出来。
这套文档现在把体验拆分得更清晰：

- 当你想用最短路径从零走到一个可用项目时，用[快速开始](/zh/getting-started/)。
- 当你的代理已经装好，并且你想走最清晰的创建项目、放入代码片段和验证第一个 page view 的路径时，用 [5 分钟完成第一个项目](/zh/guides/first-project-in-5-minutes/)。
- 当你想用提示词驱动真实任务流程或配置流程时，用这些指南。
- 当你想最清楚地理解 plugin、skill、MCP、CLI 和 API 在不同代理环境里的用法时，用 [CLI、MCP 与 API](/zh/reference/cli-mcp-api/)。
- 当你需要逐个端点查看请求和响应细节时，用 API 参考。
- 当你需要一份紧凑、可机读的文档索引时，用 LLM 文本导出。
