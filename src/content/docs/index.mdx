Agent Analytics is a web analytics platform with an agent-friendly HTTP API, an MCP server, and an agent skill layer.
These docs now split the experience cleanly:

- Use the guides when you want the fastest install path for a specific agent.
- Use the API reference when you need endpoint-level request and response detail.
- Use the LLM text exports when you want a compact machine-readable index of the docs.
