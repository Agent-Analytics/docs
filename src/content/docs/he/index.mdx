Agent Analytics היא פלטפורמת אנליטיקת ווב עם משטח אחד לסוכנים, שנחשף דרך plugin, skill, MCP, CLI ו-HTTP API.
התיעוד הזה מחלק עכשיו את החוויה בצורה נקייה:

- השתמשו ב-[התחלה מהירה](/he/getting-started/) כשאתם רוצים את המסלול הקצר ביותר מאפס לפרויקט חי.
- השתמשו ב-[הפרויקט הראשון ב-5 דקות](/he/guides/first-project-in-5-minutes/) כשהסוכן כבר מותקן ואתם רוצים את המסלול הכי ברור ליצירת פרויקט, שתילת סניפט ואימות ה-page view הראשון.
- השתמשו במדריכים כשאתם רוצים מסלול prompt-first להגדרה או ל-workflow ממוקד-משימה.
- השתמשו ב-[CLI](/he/reference/cli/) כשאתם רוצים את המסלול ה-shell-first עצמו.
- השתמשו ב-[CLI מול MCP מול API](/he/reference/cli-mcp-api/) כשאתם רוצים את ההסבר הכי ברור של plugin, skill, MCP, CLI ו-API על פני סביבות סוכן שונות.
- השתמשו בעיון ב-API כשאתם צריכים פירוט מדויק של בקשות ותגובות ברמת נקודת קצה.
- השתמשו בייצואי הטקסט ל-LLM כשאתם צריכים אינדקס קומפקטי וקריא-מכונה של התיעוד.
