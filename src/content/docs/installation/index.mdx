Choose the environment your AI agent already runs in. The analytics surface stays the same; only the native entrypoint changes. Each page leads with the recommended path for that environment, then covers alternatives where they matter.

If you are landing here from a Paperclip workflow, start with [Set up Agent Analytics for your Paperclip company](/guides/paperclip/) and create the CEO task before you pick an environment-specific install page.
