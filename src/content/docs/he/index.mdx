Agent Analytics היא פלטפורמת אנליטיקה לאתרים עם HTTP API ידידותי לסוכנים, שרת MCP ושכבת skills לסוכנים.
התיעוד הזה מחלק עכשיו את החוויה בצורה נקייה:

- השתמשו ב-[התחלה מהירה](/he/getting-started/) כשאתם רוצים את המסלול הקצר ביותר מאפס לפרויקט חי.
- השתמשו במדריכים כשאתם רוצים מסלול prompt-first להגדרה או ל-workflow ממוקד-משימה.
- השתמשו בעיון ב-API כשאתם צריכים פירוט מדויק של בקשות ותגובות ברמת נקודת קצה.
- השתמשו בייצואי הטקסט ל-LLM כשאתם צריכים אינדקס קומפקטי וקריא-מכונה של התיעוד.
