Agent Analytics gives AI agents such as Claude Code, Cursor, Codex, and OpenClaw one web analytics surface, exposed through plugin, skill, MCP, CLI, and HTTP API access paths.
These docs now split the experience cleanly:

- Use Getting Started when you want the shortest path from zero to a live project.
- Use [Set up Agent Analytics for your Paperclip company](/guides/paperclip/) when your company runs on Paperclip and you want the direct CEO-task and finish-code path.
- Use [First Project in 5 Minutes](/guides/first-project-in-5-minutes/) when your AI agent is already installed and you want the clearest create-project, place-snippet, and verify-first-page-view loop.
- Use the guides when you want a prompt-first setup or workflow for a real task.
- Use [CLI vs MCP vs API](/reference/cli-mcp-api/) when you want the clearest explanation of plugin, skill, MCP, CLI, and API across different agent environments.
- Use the API reference when you need endpoint-level request and response detail.
- Use the LLM text exports when you want a compact machine-readable index of the docs.
